I'm a web developer who can do front-end and back-end work, with a Master's in Computer Science, and work experience as an Information Technology (IT) technician.

I first started off working in the IT departments of various companies, building my skills in troubleshooting a myriad of issues, both hardware and software. I also worked on networking problems and administered servers on various platforms, including Linux and Microsoft Windows Server. I wanted to specialize in web development, and that led me to a coding boot camp called Learning Fuze where I developed the required skills to create full-stack web applications. *Features the ability to delete or edit any card

Tech Stack: Babel, Bootstrap, CSS, HTML, JavaScript, jQuery, Lodash, Node.js, React.js, Webpack

Homepage:

Metalink:
*A mobile-first, full-stack web application for lovers of metal music that aims to connect them with other fans of the genre
*Users can post comments, album reviews, or other information for others to see when they log into the application
*Bcrypt used to authenticate users and hash user passwords which are stored in a PostgreSQL table
*Users can search for YouTube videos, metal shows via the TicketMaster API, and band information via the Discogs API
*Concert search results are populated on Google Maps as clickable, animated markers with the corresponding concert information
*Profile information can be saved and edited, as well as user posts
*Registered users can join a global chat room and message each other in real-time with chat history saved upon logging out
*In case of forgotten passwords, users can have a password reset link e-mailed to them which expires in 1 hour using a token
